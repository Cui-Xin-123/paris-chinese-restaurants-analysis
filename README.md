# Analyse des restaurants chinois à Paris

## Présentation du projet

Ce projet de Data Science vise à analyser les restaurants chinois à Paris à partir de données collectées en ligne.

L'objectif est d'explorer les caractéristiques des établissements, d'étudier les avis clients, d'identifier les tendances géographiques et de mieux comprendre les préférences des consommateurs grâce à des techniques de traitement de données, de NLP et de visualisation.

---

## Objectifs

* Collecter des données de restaurants chinois via web scraping.
* Nettoyer et structurer les données collectées.
* Réaliser une analyse exploratoire des données.
* Étudier les avis clients à l'aide de techniques NLP.
* Identifier les tendances géographiques et culinaires.
* Construire un tableau de bord interactif avec Power BI.

---

## Données collectées

Les données incluent notamment :

* Nom du restaurant
* Adresse
* Arrondissement
* Type de cuisine
* Note moyenne
* Nombre d'avis
* Commentaires clients

---

## Méthodologie

### 1. Web Scraping

* Extraction des données depuis des plateformes publiques.
* Collecte automatisée des informations sur les restaurants.

### 2. Nettoyage des données

* Suppression des doublons.
* Gestion des valeurs manquantes.
* Standardisation des catégories.

### 3. Analyse exploratoire

* Distribution des notes.
* Répartition géographique des restaurants.
* Analyse du nombre d'avis.

### 4. Analyse NLP

* Extraction de mots-clés.
* Analyse de sentiments.
* Identification des thèmes récurrents dans les commentaires.

### 5. Visualisation

* Création de graphiques exploratoires avec Python.
* Construction d'un dashboard Power BI interactif.

---

---

## Exemples d'analyses

* Quels arrondissements possèdent la plus forte concentration de restaurants chinois ?
* Quels établissements obtiennent les meilleures évaluations ?
* Quels mots apparaissent le plus fréquemment dans les avis positifs et négatifs ?
* Quels types de cuisine sont les plus appréciés par les clients ?

---

## Résultats

L'analyse met en évidence des différences significatives entre les quartiers de Paris en termes de densité de restaurants, de notes moyennes et de préférences culinaires.

Les techniques NLP permettent également d'identifier les principaux facteurs influençant la satisfaction des clients.
